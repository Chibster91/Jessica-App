Design notes

  • Source filter (Add Food). Same pill segmented control as the Weight sub-tabs — three pills above the search, ordered All / Custom / Recipes. Active pill takes the purple fill; inactive pills are quiet. Each shows a tiny mono count so users know whether their list is empty before tapping in.
  • Search placeholder follows the filter. "Search 350,000+ foods" on All, "Search custom foods" on Custom, "Search recipes" on Recipes. Same input, different copy.
  • "Create" tile. One reusable component used in both the modal and the Library tabs. Dashed purple border + soft purple + glyph differentiates it from a real food row, keeps it discoverable without shouting. Title in accent purple; subtext muted. Tapping opens the existing custom-food / recipe builder.
  • Library parity. Custom and Recipes tabs each get the matching Create tile pinned at the top of the list. Recent tab is unchanged — no create affordance needed there.
  • What I did not change. The All filter still goes to the USDA-backed search. The Recents-style food rows, modal chrome, and bottom nav all stay as in the existing kit.
